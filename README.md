# Dynamic Employee Directory App

This project is a React application that dynamically fetches employee data from a public API and displays it in responsive employee cards. The application uses modern React features such as **useState** and **useEffect** to manage state and retrieve data. Users can easily search employees by **name or email**, and the interface updates automatically as they type. The project demonstrates key React concepts including **API fetching, component-based architecture, reusable components, dynamic rendering using `.map()`, and responsive UI design**.

## Features

* Fetches employee data from a public API
* Displays employees in responsive cards
* Real-time search by name or email
* Reusable `EmployeeCard` component
* Dynamic rendering using JavaScript array methods
* Responsive layout for mobile, tablet, and desktop
